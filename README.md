# Lascar-Project

LAS'CAR est une solution applicative pour covoiturer à l'instar de BLABLACAR.
Les visiteurs peuvent : consulter la page d'accueil, consulter les trajets, sans pour autant réserver.
Les membres connectés, comme un conducteur ou un passager, ont accès à plus de fonctionnalités.

L'équipe de développement qui a travaillé sur le projet est composée de 4 personnes : Alan, David, Gabin et moi-même.

Les fonctionnalités prévues :
* Page d'accueil
* S'inscrire
* Se connecter
* Proposer un trajet
* Consulter les trajets
* Consulter le détail d'un trajet
* Réserver un trajet
* Gérer son profil
* Gérer son solde
* Consulter son planning de trajets (publication et réservation)
* Paramétrer ses alertes

La phase de conception réalisée sur FIGMA inclus les éléments suivants : 
* Diagrammes UML (sur FigJam)
* Modélisation de la base de données : méthode Merise MCD/MLD/MPD (sur FigJam)
* Prototype Application LAS'CAR Web (sur Design)
